⏱️ Drag Delay

Add a delay before drag starts - useful for touch devices and preventing accidental drags

Hold to Drag (300ms delay)

🎯 Hold for 300ms to start dragging
📱 Perfect for mobile interfaces
🚀 Prevents accidental drags
✨ Allows quick taps without triggering drag
💡 Tip: Click and hold any item for 300ms to start dragging. Quick clicks won't trigger drag mode.

Configuration

new Sortable(element, { delay: 300, // 300ms delay delayOnTouchOnly: true, // Only on touch touchStartThreshold: 5, // 5px movement cancels animation: 150 }); // Useful for: // - Mobile apps with scrollable lists // - Lists with clickable items // - Preventing mis-clicks
