resortablev2.0.0-alpha.1

Modern TypeScript rewrite of Sortable.js — reorderable drag-and-drop lists with FLIP animations, multi-drag, plugins, and full accessibility.

Alpha — not ready for production use. APIs are still in flux and the library has not been published to npm. If you need a drag-and-drop library today, use SortableJS.
